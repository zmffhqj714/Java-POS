  /* 프로그램 구조 */
  *******************
       Packages
  *******************
  1. views :: View.class
  2. controllers :: Loader.class, FrontController.Class
  3. model :: MenuManagement.class >> 메뉴, 굿즈
  			  MemberManagement.class  >> 회원
              Sales.class  >> 판매
  			  StoreManagement.class >> 매장
  			  DataAccessObject.class
   4. beans

  /* 개발 패턴 :: MVC */
  View :: 화면구현 --> 입출력 구현
  Controller :: 클라이언트로부터 발생하고 서버로 전달된 데이터의 서비스 분기
  Model :: 비즈니스 로직 구현 >> Service Class
  +
  DTO, VO
  +
  Data Access Object
  +
  File DB
  
  /* Data Flow */
  Loader :: main()
  --> View.View() 
  --> ctlView() :: 서버와의 데이터 흐름 제어 :: 11
       --> int ctlMain() :: 메인화면 출력  << 1
  	   --> int ctlSub() :: 서브화면 출력 << 1
